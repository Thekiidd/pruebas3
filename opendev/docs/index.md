# OpenDev Agent Platform Documentation

Welcome to OpenDev, the open-source alternative to Claude Code.

## Features

- **Multi-Model Support**: Anthropic, OpenAI, Google Gemini, Ollama
- **Tools**: Filesystem, Shell, Web Search, Git, Browser Automation, Code Analysis
- **Interfaces**: CLI (Ink), Web (React), VS Code Extension
- **Memory**: Vector embeddings, conversation summarization
- **MCP**: Native Model Context Protocol client
